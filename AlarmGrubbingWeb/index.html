Satu Sentuhan Untuk Semua Perangkat

Bagaimana Cara Menyinkronkan Alarm?

Sinkronisasi kelompok bekerja secara instan berbasis Cloud & Pin 6-Digit Acak. Tanpa perlu log in, tanpa perlu memasukkan email, aman dan sangat privat!

1. Alur PEMBUAT (Kirim)

Membuat & Membagikan Kolektif

Langkah A: Buat Kamar Alarm Baru

Buka aplikasi Alarm Grup, lalu ketuk tombol "Buat Kamar" di panel kelompok. Tentukan jam target dan profil rujukan.

Langkah B: Dapatkan Token Unik (PIN)

Sistem Firebase realtime akan menggenerasi token 6-digit unik (Contoh: BELAJAR-101).

Langkah C: Bagikan ke Teman

Ketuk token tersebut untuk menyalin kode secara instan atau langsung bagikan tangkapan kodenya ke WhatsApp, Telegram, atau SMS kelompok Anda.

TOKEN KAMAR SINKRON BELAJAR-101
Salin Kode

2. Alur PENERIMA (Terima)

Memasukkan Token & Sinkronisasi

Langkah A: Terima Token Dari Teman

Buka aplikasi Alarm Grup di perangkat Android Anda, pilih tab kelompok, lalu ketuk menu "Join Kamar".

Langkah B: Tempel & Konfirmasi PIN

Tempelkan kode yang disalin (Contoh: BELAJAR-101) lalu tekan tombol konfirmasi terima.

Langkah C: Sinkron Instan Berhasil

Alarm secara otomatis tersimpan di perangkat lokal Anda. Saat pembuat alarm mengubah jadwal, alarm Anda otomatis ikut terupdate dalam hitungan milidetik!
