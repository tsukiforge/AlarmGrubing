Kebijakan Privasi (Privacy Policy)

ID Paket Aplikasi: com.kukkistudio.alarmgrup

Pembaruan Terakhir: 12 Juni 2026

Terima kasih telah menggunakan aplikasi Alarm Grup yang dikembangkan oleh Kukki Studio. Kami sangat menghargai kepercayaan Anda dan berkomitmen untuk melindungi privasi serta keamanan data pribadi Anda. Halaman kebijakan privasi ini menjelaskan bagaimana aplikasi kami mengelola data Anda.

1. Pengumpulan & Penggunaan Data

Aplikasi Alarm Grup beroperasi dengan pendekatan Offline-First & Local-First untuk menjamin privasi Anda secara maksimal:

  • Foto Profil: Foto profil kustom yang Anda pilih disimpan secara eksklusif dalam direktori sandboxed lokal perangkat Anda (context.filesDir) dan tidak pernah diunggah ke server mana pun.
  • Pemusatan Musik/MP3 Kustom: Akses ke file audio / musik Anda semata-mata digunakan untuk memainkan nada dering alarm yang Anda pilih di perangkat lokal Anda.
  • Sinkronisasi Kamar (Group Lobbies): Saat Anda menggunakan fitur alarm kelompok, informasi mengenai nama grup, kode kamar, dan status alarm Anda tersinkronisasi murni secara real-time melalui enkripsi dan protokol real-time Firebase dengan aman. Data ini hanya digunakan demi keberfungsian fitur grup yang bersangkutan.

2. Izin Perangkat (In-App Permissions)

Kami memerlukan beberapa izin sistem Android yang penting untuk menjalankan fungsionalitas utama aplikasi:

  • Notification Access: Untuk menampilkan status bar informasi, pemberitahuan waktu alarm berikutnya, dan trigger alarm rilis.
  • Exact Alarm: Digunakan untuk merespons trigger waktu alarm secara menit-terinci agar tepat sasaran waktu bahkan saat ponsel dalam keadaan tertidur (doze mode).
  • Overlay Window (System Alert Screen): Untuk menyajikan tampilan dering penuh (immersive ringing screens) yang modern agar alarm Anda segera dimatikan atau disnooze tanpa harus entsirkular kunci layar secara rumit.

3. Keamanan Data & Pihak Ketiga

Kami tidak melakukan transaksi, jual-beli, pemindaian, ataupun komparasi data privasi Anda dengan biro analitik bayaran atau pihak periklanan pengganggu manapun. Komunikasi API hibrida kami telah dimodifikasi kokoh agar aman dari enkripsi sewenang-wenang maupun pemblokiran oleh seluruh ISP (Internet Service Provider) utama di Indonesia guna kelangsungan sinkronisasi lobi grup Anda.

4. Perubahan Kebijakan Privasi

Kami dapat memperbarui kebijakan privasi ini dari waktu ke waktu guna menyelaraskan dengan pembaruan platform Android maupun peningkatan kepatuhan play store. Langkah penyesuaian akan diumumkan melalui pembaruan di halaman situs web atau rilis perubahan deskripsi build aplikasi.
